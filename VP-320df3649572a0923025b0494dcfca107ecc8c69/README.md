## Lab 3

- H2 Setup:

Create a new database in H2 Console: `jdbc:h2:~/web`

`create table author(id bigint generated by default as identity primary key, name varchar(255), surname varchar(255), country varchar(255), biography varchar(255));`

`create table book (id bigint generated by default as identity primary key, title varchar(255),genre varchar(255), country varchar(255), average_rating double precision, author_id bigint references author);`

`alter table author add column if not exists book_ids bigint references book on delete cascade`
